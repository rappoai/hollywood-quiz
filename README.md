# hollywood-quiz
A simple web-based hollywood quiz!

## Build and run (development)

### Gitpod
Open a terminal and run:
```
npx browser-sync -w
```

### Local machine
Install node (tested with v16.13.2), then run:
```
npx browser-sync -w
```

## Deploy (production)
Use [netlify](https://netlify.com) to quickly deploy this website.
